# Connor Review

Local PR review tool. See [the design spec](docs/superpowers/specs/2026-05-20-connor-review-design.md) and [implementation plan](docs/superpowers/plans/2026-05-20-connor-review.md).

## Run

```bash
npm run install:all
npm run dev
```

Then open http://localhost:5173. Requires `gh auth login` to be active.
