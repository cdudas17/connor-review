Next-meeting FAB — design sandbox

Live preview of every state the floating widget can be in. Edit this file (or the controls below) and reload to iterate. When something looks right, tell Claude and we'll port the change into web/src/components/NextMeetingFab.tsx + web/src/styles/app.css.

States

live — you're currently in a meeting. Default colour: red. Shows time until the event ends.
soon — next meeting is within the "soon" cutoff (default 5 min). Default colour: amber.
idle — routine countdown to the next meeting. Default colour: accent blue.
free — no more events today. Default colour: dim green, 85% opacity.

Tweak parameters

Custom colours

Soon uses an animated rainbow gradient (no flat colour). Edit the CSS at the top of this file to tweak the gradient stops or animation timing.

Changes here only affect this preview page. When you've landed on something you like, ping Claude with what to apply (e.g. "make Soon cutoff 10 minutes and pulse when live").
